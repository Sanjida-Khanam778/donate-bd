Important FAQ

What is the Document Object Model (DOM)?

DOM is a model which is used for interacting the web page using JS. Here Document means any file like .html, .css etc. These files are represented as objects in JS and DOM is a tree model.

How do you select an element from the DOM?

We can select an element from the DOM by using some element selector. Multiple element selector is used for selecting multiple element at a time whereas single element selector are used for selecting only one element.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique for handling event efficiently. Here more than one event is handled by using only one event listener. If we add event listener to parent element then the event is handled with all the child element. Furthermore, event is handled over a newly added child in event delegation. For these reason, event delegation is useful.

How do you manipulate an element's attributes and styles using the DOM?

For manipulating element's attributes and styles, we have to use some method. First we have to select the element and then apply the methods on it. For manipulating attributes, getAttribute(), serAttributes(), and removeAttributes are used. For applying style, classList() and .style is used.
